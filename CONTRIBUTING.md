# Katkıda Bulunma

COSA Home Assistant entegrasyonuna katkıda bulunmak istediğiniz için teşekkürler!

## Hata Bildirimi

1. GitHub Issues'da benzer bir hata olup olmadığını kontrol edin
2. Yeni bir issue açın ve şunları belirtin:
   - Home Assistant versiyonu
   - COSA entegrasyon versiyonu
   - Hata mesajı (log dosyasından)
   - Hatayı tekrarlamak için adımlar

## Geliştirme

1. Bu repoyu fork edin
2. Yeni bir branch oluşturun (`git checkout -b feature/yeni-ozellik`)
3. Değişikliklerinizi commit edin (`git commit -am 'Yeni özellik eklendi'`)
4. Branch'inizi push edin (`git push origin feature/yeni-ozellik`)
5. Pull Request açın

## Kod Standartları

- Python 3.9+ uyumlu kod yazın
- Type hints kullanın
- Docstring ekleyin
- HACS ve Hassfest validasyonlarını geçtiğinden emin olun

## Lisans

Katkılarınız MIT lisansı altında yayınlanacaktır.
